# Hestia - Proyecto Fundación Pescar

Este es el repositorio base para el backend del proyecto Hestia.

## Cómo empezar

Sigue estos pasos para configurar el proyecto de manera local

## 1. Clonar el repositorio

```sh
git clone https://github.com/DanyGeier/hestia-back.git
```

## 2. Instalación
Entra en la carpeta del backend e instala las dependencias necesarias:

```sh
cd hestia-back
npm install
```

## 3. Ejecución
Para iniciar el servidor en modo desarrollo:

```sh
npm run dev
```

El servidor estará disponible en 'http://localhost:3000' 

## Tecnologías utilizadas
* [Node.js] (https://nodejs.org) - Entorno de  ejecución
* [Nodemon] (https://nodemon.io) - Herramienta de desarrollo
